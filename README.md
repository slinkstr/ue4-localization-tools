# ue4-localization-tools
Forked from majomix/ue4-localization-tools  
Use to import/export .locres files  
Forked just for the readme  
```
Usage: UE4TextConverter [OPTIONS]
Options:
	--locres=VALUE         
	--txt=VALUE            
	--export               
	--import               
	--compact              
	--sort                 
	--close
```
